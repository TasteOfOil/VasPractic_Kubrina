Проект: 1-memory-type

Краткое описание: Программа демонстрирует работу с тремя видами памяти: 
			локальной, статической, динамической.

Описание: Программа считывает введенное пользователем число N.
		Далее демонстрируется работа с несколькими видами памяти:
		-- Статический:
			В цикле N раз вызывается функция, имеющая статическую переменную.
			В вызываемой функции происходит инициализация статической переменной,
			вывод переменной в терминал и увеличесние значения на 1.
		-- Локальной:
			В цикле N раз вызывается функция, содержащая локальную переменную.
			В вызываемой функции переменная инициализируется 0, выводится в терминал
			и увеличивается на 1.
		-- Динамической:
			После ввода пользователем числа N выделяется память для динамического массива 
			размером N. Этот массив заполняется значениями (в обратном порядке), 
			которые возвращает функция со статической переменной.

Сборка: Cборка осуществляется с помощью CMake
	Файл 1-memory-type.cpp и CMakeLists.txt должны находиться в одном каталоге.
	Для сборки и запуска проекта необходимо написать следующие команды в терминал:
	cmake <путь к каталогу с файлами проекта>
	make
	./prog1
